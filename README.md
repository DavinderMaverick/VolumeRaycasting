## Usage
Press 1 -> Transfer Function Only  
Press 2 -> Shading Enabled  
Esc -> Quit  

Mouse Click and Drag -> Rotate View  
Mouse Scroll Wheel -> Zoom In/Out  

## Project Dependencies
1. opengl  
2. glad  
3. glfw  
4. glm  
5. stb
